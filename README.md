# Moonshine Website

Simple website for a music band, built with Next.js.

## Technologies

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4
- ESLint

## Requirements

- Node.js
- npm

## Installation

Install dependencies:

```bash
npm install
```

## Local Development

Start the development server:

```bash
npm run dev
```

The website will be available at:

```text
http://localhost:3000
```

## Production Build

Build the project:

```bash
npm run build
```
